<Section icon={Activity} title="構えのメカニクス：ユナイテッドステップとスピナーボール">

**鉄拳8のボブの格闘スタイル**を支える重要な柱の一つが、2つの万能な構えです。これらの構えにより、ボブはニュートラル状態からシームレスに強力な二択（揺さぶり）へ移行できます。2026年のシーズン3アップデートでは、これらの構えに回復可能ゲージ（リカバリアブルヘルス）システムと相互作用する新しい特性が追加されました。

**ユナイテッドステップ**は、依然としてボブの主要なプレッシャーツールです。**ラングワッシャー**や**カッティングコッパ**といった代表的な技へのアクセスを維持しながら、相手との距離を詰めることができます。これらの技は相手の動きを制限するように設計されており、高速の下段か、強力な中段かの二択を強要します。

**スピナーボール**は、一方、『鉄拳8』で大幅な強化を受けました。空中コンボの繋ぎとして使用できるようになり、ボブの壁運び能力はかつてないほど恐るべきものになっています。さらに重要なのは、スピナーボールがヒートシステムへの入り口となった点です。

| 構え名 | 主要技 | 戦闘での役割 |
| :--- | :--- | :--- |
| **ユナイテッドステップ** | ラングワッシャー、カッティングコッパ | プレッシャー、中距離からの接近 |
| **スピナーボール** | ルーレットカッター、ヒートエンゲイジャー | コンボの拡張、ヒート発動 |
| **パーフェクトボディ** | 当身（アタックリバーサル） | 防御的カウンター、主導権のリセット |
| **おにぎりボンバー** | 高ダメージ打撃 | ガード崩し、ストーリー関連のフィニッシャー |

<Callout type="warning" title="注意">
スピナーボールは俊敏ですが、発生の早いジャブや中段の牽制技で割り込まれる可能性があります。布石を打たずに遠距離からワンパターンに繰り出すのは避けましょう。
</Callout>

</Section>

<Section icon={Zap} title="ヒートシステムとの連携とジャストフレーム・スマッシュ">

ヒートシステムこそ、2026年のメタにおいて**鉄拳8のボブの格闘スタイル**が真に輝く要素です。ボブのヒートエンゲイジャーは、特定のスピナーボールからの派生など、すでに彼の立ち回りの核心となっている技に戦略的に配置されています。ボブがヒート状態に入ると、地上戦の能力を強化するツールが解放され、元々難易度の高かった確定反撃の実行を補いやすくなります。

ヒート状態における最大の目玉は、**ジャストフレーム・ヒートスマッシュ**です。単にボタンを押すだけで発動する多くのキャラクターのヒートスマッシュとは異なり、ボブのプレイヤーは特定のタイミングで追加入力を行うことで、モーションを変化させることができます。これにより、ダメージが大幅に増加するだけでなく、相手の回復可能ゲージをより多く削り取ることができます。

| ヒートツール | 効果 | 戦略的用途 |
| :--- | :--- | :--- |
| **ヒートバースト** | 標準的な発動打撃 | 相手の攻勢を止めるために使用 |
| **ヒートエンゲイジャー** | ヒット時にボブが前方に突進 | 近距離の二択（揺さぶり）へ移行 |
| **ヒートダッシュ** | ヒートを消費してコンボを伸ばす | 浮かせ技の後にダメージを最大化 |
| **ヒートスマッシュ** | ジャストフレーム入力でダメージ増加 | ラウンドを終わらせるための切り札 |

<Callout type="success" title="成功の秘訣">
プラクティスモードでヒートスマッシュのタイミングを練習しましょう。ボブのグローブが特定の発光をするのが視覚的な合図です。これを安定して決められるかどうかが、大逆転と敗北の分かれ目になります。
</Callout>

</Section>

<Section icon={Target} title="ステップバイステップガイド：ボブの基本をマスターする">

  <Step num="1" title="「スピード＆ウェイト」のリズムを覚える">

  </Step>
  <Step num="2" title="構えの移行をマスターする">

  </Step>
  <Step num="3" title="基本的な確定反撃を決める">

  </Step>
  <Step num="4" title="ヒートエンゲイジャーを組み込む">
