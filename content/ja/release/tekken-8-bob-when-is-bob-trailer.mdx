<Section icon={Target} title="構えと技リストの解説">

ボブのゲームプレイの核となるのは、2つの主要な構えである**「ユナイテッドステップ」**と**「スピナーボール」**です。鉄拳8では、これらの構えがヒートシステムと統合される形で拡張され、相手にガードされている状況でもプレッシャーを維持する新たな手段が提供されています。

### 主要な構え
- **ユナイテッドステップ**: 前方に移動するボブの特徴的な構えです。一瞬で間合いを詰めることができます。復活した**「ラングワッシャー」**や**「カッティング・コッパ」**などの技は、相手の動きを制限し、高ダメージの浮かせ技を狙う上で引き続き不可欠です。
- **スピナーボール**: 空中コンボでも使用可能になった回転する構えです。新たな**ヒートエンゲージャー**の選択肢が追加され、ボブの攻撃的な流れの中心的なハブとなっています。

| 技名 | タイプ | 説明 |
| :--- | :--- | :--- |
| **おにぎりボマー** | 新技 | 新たなストーリーテーマに関連した強力な打撃技。 |
| **ギガジャッカー** | パワークラッシュ | 戦況を覆すための、発生の早いアーマー付きの技。 |
| **ルーレットカッター** | 構え攻撃 | ガード時の圧力を維持する、スピナーボールからの拡張された選択肢。 |
| **パーフェクトボディ** | 当て身技 | 相手の打撃を受け流す新たな防御メカニクス。 |

<Callout type="info" title="構えの連携フロー">
いくつかの下段攻撃からスピナーボールへ移行できるようになりました。これにより、防御を固める相手を崩し、即座に構えからのプレッシャーやヒートダッシュによるコンボ移行へと繋げることができます。
</Callout>

</Section>

<Section icon={Sparkles} title="ヒートシステムとレイジの統合">

鉄拳8で導入されたヒートシステムは、ボブに大きな恩恵をもたらします。彼のヒート状態は地上戦を強化し、高い技術を持つプレイヤーに独自の恩恵を提供します。

**ヒートシステムの特徴:**
- **ヒートエンゲージャー**: スピナーボールからの派生技などの技は、ヒット時にヒートを誘発し、大きなフレーム有利を持った状態でボブを前方に突進させます。
- **ヒートダッシュ**: ヒートエンゲージャーの後、ボブはゲージを消費してコンボを継続したり、ガード時に二択を迫ったりすることができます。
- **ジャストフレーム・ヒートスマッシュ**: ボブのヒートスマッシュは独特です。最後の入力を完璧なタイミング（ジャストフレーム）で行うと、ダメージが増加し、通常とは異なるシネマティックな特別演出が発生します。

<Callout type="success" title="シーズン3のシステム変更点">
シーズン3において、ヒートスマッシュの壁やられ（壁張り付き）の挙動が調整されたことに注意してください。ボブ使いのプレイヤーは、壁やられを狙うためにヒートスマッシュだけに頼るのではなく、壁運びのためにヒートダッシュを活用することに焦点を当てるべきです。
</Callout>

| メカニクス | メリット | 使用のコツ |
| :--- | :--- | :--- |
| **ヒートバースト** | 共通の中段攻撃 | 相手のアグレッシブな連携を途中で止めるために使用。 |
| **ヒートスマッシュ** | 高ダメージのフィニッシャー | 最大火力を出すためにジャストフレームのタイミングを狙う。 |
| **レイジアーツ** | シネマティックな一発逆転技 | 体力が低い時に、回復可能ゲージ（青体力）を消し去るのに非常に効果的。 |

</Section>

<Section icon={Shield} title="初心者向けボブ攻略ガイド">

このキャラクターを初めて使う方や、久しぶりに復帰する方は、段階的な上達プロセスを踏むことで、彼の「スピード＆ウェイト」スタイルを効果的にマスターすることができます。

<Steps>
  <Step num="1" title="基本を学ぶ">
    まずはボブの基本的な牽制（ポーク）と移動の練習から始めましょう。その体型にかかわらず、高機動キャラクターのように操作すべきであることを理解してください。近〜中距離のツールに焦点を当て、試合のペースをコントロールしましょう。
  </Step>
  <Step num="2" title="構えの移行をマスターする">
    通常の連携技から**「ユナイテッドステップ」**や**「スピナーボール」**へ移行する練習をしましょう。これらの構えにシームレスに移行できるようになることが、ボブのプレッシャーを相手にとって脅威にする鍵です。
  </Step>
  <Step num="3" title="ヒートの活用を最適化する">
    どの技が**ヒートエンゲージャー**であるかを覚えましょう。接戦のラウンドでは追加ダメージが勝敗を分けるため、プラクティスモードでジャストフレーム・ヒートスマッシュのタイミングを筋肉が覚えるまで練習してください。
  </Step>
  <Step num="4" title="基本コンボを決める">
    安定した空中コンボからの運びに集中しましょう。ボブは優れた壁運び能力を持っています。コンボを壁際で終わらせ、そこから彼の強力な起き攻め（地上戦）を展開する方法を学びましょう。
  </Step>
</Steps>

鉄拳8におけるボブの物語は、英雄的でありながらどこかコミカルな展開を見せます。ブライアン・フューリーが絡んだ事件の後、ボブは回復し、巨大なバーガー帝国を築き上げました。しかし、彼の正義への情熱は、アメリカで社会現象となるフードチェーン**「ボブのおにぎり（Bob's Onigiri）」**を立ち上げるきっかけとなりました。

**おにぎり救世主:**
G社がニューヨークに侵攻した際、ボブは自身の資産を投げ打って市民に食料支援を行いました。これにより、彼は「おにぎり救世主（Onigiri Savior）」というニックネームで呼ばれるようになります。ストーリーモードでの彼の葛藤は、マンハッタン地方検事局の臨時捜査官としての新たな役割に焦点を当てており、三島一八を法の裁きにかける任務に挑みます。

**公開されたコスチューム:**
- **コスチューム1 (ストリートカジュアル)**: 「ボブのおにぎり」のブランディングが施された、リラックスしたトレーニングウェア。この衣装は『鉄拳タッグトーナメント2』のスリムボブの見た目を彷彿とさせますが、鉄拳8の「パーフェクトボディ」に合わせて調整されています。
- **コスチューム3 (捜査官フォーマル)**: 地方検事局での彼の公的な役割を反映した、シャープなフォーマルスーツ。これまでの作品では滅多に見られなかったプロフェッショナルな外見を彼に与えています。
